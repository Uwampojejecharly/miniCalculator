# miniCalculator

This project involved creating a minicalculator using Java and the librairie Swing.

## Project Structure

The project is divided into three main classes:

1.CalculatorUI: This class manages the user interface of the calculator.

2.CalculatorHelper: This class contains methods to perform calculations.

3.Listeners: This class handles user interactions with the calculator.

## How to Use the Project

To use this project, you can follow these steps:

1.Clone the repository from GitHub.

2.Open the project in your favorite IDE.

3.Run the program from the main class.
